什麼是「多模態」？

AI 同時看圖 + 讀字

一般翻譯只吃「文字」。多模態（multimodal）模型能同時理解「圖片」和「文字」——所以它能看懂照片裡的菜單、招牌。

Gemini 的 gemini-3.5-flash 就支援圖片與 PDF，我們直接把影像丟給它。

提示詞

要求輸出 JSON：摘要 + 翻譯
幫我做拍照翻譯：後端 analyze() 收到圖片 bytes，
用 Gemini 多模態讀圖，輸出 JSON {summary, translation}：
summary=用目標語言摘要重點，translation=完整翻譯。
關閉 thinking（避免思考 token 吃掉輸出導致 JSON 被截斷），
並做 JSON 解析容錯。再做 /api/vision 端點收 base64 影像。

後端 analyze()（重點）

多模態讀圖 → JSON
# providers.py
# text / file_bytes 擇一：圖片與 PDF 用 file_bytes，純文字用 text（CH12 會用到）
def analyze(api_key, target_name, text=None, file_bytes=None, mime_type=None):
    from google import genai
    from google.genai import types
    client = genai.Client(api_key=api_key)
    system = (f"Read the input. Write a concise summary in {target_name}, "
              f"and a faithful full translation into {target_name}. "
              'Respond ONLY as JSON: {"summary":"...","translation":"..."}')
    parts = []
    if file_bytes is not None:
        parts.append(types.Part.from_bytes(data=file_bytes, mime_type=mime_type))
    if text:
        parts.append(text)
    resp = client.models.generate_content(
        model='gemini-3.5-flash',
        contents=parts,
        config=types.GenerateContentConfig(system_instruction=system,
            response_mime_type='application/json', max_output_tokens=8192,
            thinking_config=types.ThinkingConfig(thinking_budget=0)))   # 關閉思考
    return parse_json(resp.text)   # 容錯解析 summary/translation

踩過的雷：JSON 被截斷

為什麼要關 thinking

Gemini 3.x flash 預設會「思考」，思考 token 會占用輸出額度。長內容時，答案的 JSON 會被截斷（少了結尾大括號），解析失敗。

✅ 解法：thinking_budget=0 關閉思考（OCR/翻譯不需推理），輸出穩定又更快。再加「容錯解析」救回被截斷的內容。

後端 /api/vision

收 base64 影像
# app.py
import base64
@app.route('/api/vision', methods=['POST'])
def api_vision():
    data = request.get_json(force=True) or {}
    img = data.get('image','')            # dataURL
    target = data.get('target','Traditional Chinese (Taiwan)')
    mime='image/jpeg'
    if img.startswith('data:'):
        head,img = img.split(',',1); mime = head.split(':')[1].split(';')[0]
    raw = base64.b64decode(img)
    key = os.getenv('GEMINI_API_KEY') or os.getenv('GOOGLE_API_KEY')
    res = providers.analyze(key, target, file_bytes=raw, mime_type=mime)
    return jsonify({'ok':True, **res})

前端：縮圖 + 送出

先縮小省流量，再打 /api/vision
cam.onchange = async (e)=>{
  const file = e.target.files[0]; if(!file) return;
  const dataURL = await downscale(file, 1568);   // canvas 縮到最長邊 1568
  const res = await fetch('/api/vision', {method:'POST',
    headers:{'Content-Type':'application/json'},
    body: JSON.stringify({ image:dataURL, target:'Traditional Chinese (Taiwan)' })});
  const d = await res.json();
  showResult(d.summary, d.translation);   // 顯示摘要 + 翻譯
};
