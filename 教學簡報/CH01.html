一鍵安裝：怎麼用？

三步驟，跟著做
  1. 方法一（推薦）：貼一行指令——照上一頁複製 Windows / Mac 那一行，貼進 PowerShell / 終端機 按 Enter，就會自動開始裝。出現權限視窗按「是」／輸入 Mac 密碼即可。這個不會被系統擋。
  2. 方法二：下載 zip 檔——解壓後 Windows 雙擊 liangclaw.bat、Mac 把 install_ai_tools_mac.command 拖進終端機按 Enter。
🛡️ 下載的檔案被系統擋住？（很常見，別怕）
🪟 Windows 跳「已保護你的電腦」→ 點「其他資訊」→「仍要執行」；或右鍵 .bat →「內容」→勾「解除封鎖」→確定。
🍎 Mac 跳「來自未識別的開發者」→ 對檔案按右鍵 →「打開」→ 再按「打開」（或系統設定 → 隱私權與安全性 →「仍要打開」）。
👉 覺得麻煩就用「方法一：貼一行指令」，完全不會被擋。
✅ 裝好了嗎？重開終端機打 node --version、python --version、git --version、code --version、claude --version、codex --version、agy --version，有版本號就成功。
⚠️ Windows 需內建 winget（Win10/11 通常有）；找不到就到 Microsoft Store 更新「應用程式安裝程式」。Mac 用 Homebrew（沒裝會自動先幫你裝，可能要輸入 Mac 密碼）。
💡 這一步只裝「工具」。每個專案自己的 Python 套件（Flask 等）之後用 pip install -r requirements.txt 安裝，CH04 會教——別擔心現在還沒裝到。

為什麼需要 Python？

先懂它在專案裡的角色

我們的 App 分成兩半：前端（手機上看到的畫面）和後端（背後幫你處理翻譯、查天氣的伺服器）。

後端我們用 Python 這個程式語言 + Flask 這個框架來寫。Python 好讀、好學、AI 也超會寫，是初學者的首選。

🎯 你不用「學會」Python 才能開始——AI 會幫你寫，你只要看得懂大概、會執行就好。

安裝 Python（Windows）

照著點，5 分鐘搞定
  1. 打開瀏覽器，前往官方網站 python.org/downloads。
  2. 點黃色大按鈕「Download Python 3.x.x」下載安裝檔。
  3. 執行安裝檔，務必先勾選最下面的「Add python.exe to PATH」（超重要！）。
  4. 點「Install Now」，等它跑完，看到「Setup was successful」即完成。
⚠️ 沒勾「Add to PATH」的話，之後在終端機會找不到 python 指令。忘了勾就重新執行安裝檔、選 Modify 補勾。

安裝 Python（Mac）

Mac 也很簡單
  1. 前往 python.org/downloads，它會自動偵測到 macOS。
  2. 下載「macOS 64-bit universal2 installer」。
  3. 打開下載的 .pkg 檔，一路按「繼續 → 同意 → 安裝」。
  4. 輸入電腦開機密碼授權，完成安裝。
🍎 進階者也可用 Homebrew：brew install python。新手照上面的圖形安裝即可。

先認識「終端機」

用打字指揮電腦的黑視窗

終端機（Terminal）是一個用「打指令」操作電腦的視窗。之後我們會用它來執行 Python、安裝套件、啟動網站。

  • Windows：開「PowerShell」或「終端機」。
  • Mac：開「終端機 / Terminal」。
🖥️ 終端機長這樣
$ 你在這裡打指令
電腦的回覆會顯示在這裡

驗證 Python 裝好了沒

打一行指令確認

打開終端機，輸入這行，按 Enter：

python --version
Python 3.12.x   ← 有看到版本號就成功了！

若顯示「找不到指令」或版本是 2.x：Windows 試 py --version；Mac 試 python3 --version。

安裝編輯器 VS Code

最多人用、免費的程式編輯器
  1. 前往 code.visualstudio.com。
  2. 點大按鈕下載對應系統版本（會自動偵測 Windows／Mac）。
  3. 執行安裝檔，一路下一步（Windows 建議勾「加入 PATH」「加入右鍵選單」）。
  4. 安裝完打開它，看到藍色歡迎畫面就成功。
📝 編輯器就是你「看程式碼、貼程式碼、存檔」的地方。AI 生的程式碼會貼到這裡。

建立你的專案資料夾

給專案一個家
  1. 在桌面或文件夾，新建一個資料夾，命名 my-translator（英文、不要空格）。
  2. 打開 VS Code → File → Open Folder → 選這個資料夾。
  3. 之後所有檔案（app.py 等）都會放在這裡。
🖥️ 或用指令建立
mkdir my-translator
cd my-translator
code .
（最後一行會用 VS Code 打開這個資料夾）

認識 pip 與套件

幫 Python 安裝別人寫好的工具

套件（package）是別人寫好、可以直接拿來用的工具包。例如 Flask 就是一個套件。

pip 是 Python 的套件安裝器，用它一行指令就能裝好需要的工具：

pip install flask
Successfully installed flask...  ← 看到這行就裝好了

之後我們會把所有需要的套件寫在 requirements.txt，一次全部安裝，不用一個一個裝。

  • VS Code 已安裝，能開啟資料夾、能開內建終端機。
  • 選好一個 AI 助手（Claude 或 ChatGPT），能登入對話。
  • 建好專案資料夾 my-translator，並用 VS Code 打開了。

常見安裝問題

卡住先看這頁

找不到 python 指令？

Windows 沒勾 Add to PATH。重跑安裝檔選 Modify 補勾，或改用 py。

Mac 打 python 是 2.x？

改用 python3 和 pip3。

pip 不是內部指令？

試 python -m pip install ...。

公司/學校電腦裝不了？

可能有權限限制，改用個人電腦，或線上環境（如 Replit）。
