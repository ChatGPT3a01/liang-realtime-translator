踩過的雷：中文地名查不到

最重要的一課

Open-Meteo 的地理編碼對中文地名很不準：那霸、台北、大阪常常查不到！對台灣使用者是大問題。

✅ 解法：三段式地理編碼 —— ①內建熱門地點字典（保證那霸=沖繩正確）→ ②Nominatim（OpenStreetMap，中文佳）→ ③Open-Meteo（最後備援）。

三段式地理編碼

內建字典 → Nominatim → Open-Meteo
# providers/app：地名 → (lat, lon, 顯示名)
COMMON = {'那霸':(26.21,127.68,'那霸，日本'),'台北':(25.03,121.57,'台北，台灣'),
          '東京':(35.69,139.69,'東京，日本'), ...}   # 熱門地點先命中

def geocode(place):
    if place in COMMON: return COMMON[place]
    # Nominatim（要帶 User-Agent，中文支援佳）
    r = requests.get('https://nominatim.openstreetmap.org/search',
        params={'q':place,'format':'json','limit':1,'accept-language':'zh-TW'},
        headers={'User-Agent':'liang-translator/1.0'}, timeout=10).json()
    if r: return float(r[0]['lat']), float(r[0]['lon']), r[0]['display_name'].split(',')[0]
    # 最後備援：Open-Meteo geocoding
    ...

取天氣 + 帶傘建議

Open-Meteo forecast
w = requests.get('https://api.open-meteo.com/v1/forecast', params={
    'latitude':lat, 'longitude':lon,
    'current':'temperature_2m,relative_humidity_2m,weather_code,precipitation',
    'daily':'precipitation_probability_max', 'timezone':'auto','forecast_days':1}).json()

def advice(code, temp, pop):
    if code in RAIN_CODES or (pop and pop>=50): return '☂️ 建議帶傘'
    if temp>=30: return '🧴 高溫，注意防曬補水'
    if temp<=12: return '🧥 偏冷，記得保暖'
    return '👍 天氣舒適，玩得開心'

前端：查詢 + 定位

打地名，或用目前位置
// 打地名查
async function getWeather(place){
  const res = await fetch('/api/weather', {method:'POST',
    headers:{'Content-Type':'application/json'}, body:JSON.stringify({place})});
  const d = await res.json();
  show(`${d.place} ${Math.round(d.temp)}°C ${d.desc}｜${d.advice}`);
}
// 用目前位置（瀏覽器定位）
navigator.geolocation.getCurrentPosition(p=>
  fetch('/api/weather',{method:'POST',headers:{'Content-Type':'application/json'},
    body:JSON.stringify({lat:p.coords.latitude, lon:p.coords.longitude})}));
