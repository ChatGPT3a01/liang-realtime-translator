三種檔案，怎麼處理

Gemini 原生就能讀圖與 PDF
檔案類型做法
圖片 jpg/png當影像丟給 Gemini（同拍照）
PDFGemini 原生讀取，當 application/pdf 丟進去
文字檔 txt直接讀文字內容，當純文字翻譯

提示詞

用 multipart 上傳、依類型分流
做 /api/file：用 multipart 接收上傳檔案，
依副檔名/MIME 分流：txt→讀文字、pdf→當 application/pdf、
圖片→當影像，都交給 analyze() 回 {summary, translation}。
限制 15MB。前端用 <input type=file> + FormData 上傳。

後端 /api/file

依類型分流
# app.py
@app.route('/api/file', methods=['POST'])
def api_file():
    f = request.files.get('file')
    target = request.form.get('target','Traditional Chinese (Taiwan)')
    raw = f.read()
    if len(raw) > 15*1024*1024: return jsonify({'ok':False,'error':'檔案過大'}),400
    mime = (f.mimetype or '').lower()
    ext = f.filename.rsplit('.',1)[-1].lower()
    key = os.getenv('GEMINI_API_KEY') or os.getenv('GOOGLE_API_KEY')
    if mime=='text/plain' or ext in ('txt','md','csv'):
        res = providers.analyze(key, target, text=raw.decode('utf-8','replace'))
    elif mime=='application/pdf' or ext=='pdf':
        res = providers.analyze(key, target, file_bytes=raw, mime_type='application/pdf')
    else:  # 圖片
        res = providers.analyze(key, target, file_bytes=raw, mime_type=mime or 'image/jpeg')
    return jsonify({'ok':True, **res})

前端：上傳

FormData 送檔
<input type="file" id="fileIn" accept="image/*,application/pdf,text/plain,.pdf,.txt" hidden>
<button onclick="fileIn.click()">📎 上傳檔案</button>

fileIn.onchange = async (e)=>{
  const file = e.target.files[0]; if(!file) return;
  const fd = new FormData();
  fd.append('file', file);
  fd.append('target', 'Traditional Chinese (Taiwan)');
  const res = await fetch('/api/file', {method:'POST', body:fd});   // 不用設 Content-Type
  const d = await res.json();
  showResult(d.summary, d.translation);
};
