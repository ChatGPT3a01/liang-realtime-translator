這一章要做出什麼

兩人隔著手機對話

把手機平放桌上，畫面上下對開：對面的人看倒過來的上半、你看正的下半。各自講各自的語言，翻譯顯示在對方那一側。

🎁 本章小成品

一個真正「面對面」的翻譯機：適合出國問路、店家溝通。

設計概念

上半翻轉 180 度
┌─────────────┐
│  對面的人看   │  ← 上半，旋轉 180°（給對面看）
│ （翻轉的文字）│
├─────────────┤
│   你看的     │  ← 下半，正常方向
│（正常的文字）│
└─────────────┘
手機平放，兩人各據一邊

提示詞

把版面與行為講清楚
幫我做「面對面模式」：畫面上下兩塊，各有語言選單、
結果框、麥克風鈕。上半用 CSS 旋轉 180 度給對面的人看。
下方講話 → 翻成上方語言 → 顯示在上方；反之亦然。
沿用現有的 translate() 與語音辨識。給我 HTML/CSS/JS。

HTML：上下兩塊

結構
<!-- 面對面 -->
<section id="faceView">
  <div class="face top">

CSS：把上半翻過來

一行 transform 搞定
.face{ flex:1; padding:16px; }
.face.top{ transform: rotate(180deg); }   /* 給對面的人看 */
#faceView{ display:flex; flex-direction:column; height:100dvh; }

就這麼簡單——上半整塊轉 180 度，對面的人看就是正的。

邏輯：翻到對側顯示

下方說 → 上方看
// 下方(你)講 → 翻成上方語言 → 顯示在上方框
function makeSide(langSel, myBox, otherBox, otherLang){
  const rec = new Recognizer(BCP[langSel.value], async (text)=>{
    const out = await translate(text, NAME[langSel.value], NAME[otherLang.value]);
    otherBox.textContent = out;              // 顯示在對方那側
    speak(out, BCP[otherLang.value]);
  });
  return rec;
}
makeSide(f_langBottom, f_resBottom, f_resTop, f_langTop);
makeSide(f_langTop,    f_resTop,    f_resBottom, f_langBottom);
