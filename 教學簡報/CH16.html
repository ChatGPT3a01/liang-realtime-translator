讓網站能像 App 一樣裝進手機

加兩個東西：manifest（App 的名稱、圖示）和 Service Worker（背景腳本、快取），網站就能「加到主畫面」、全螢幕開、支援離線。

🎁 本章小成品

手機打開你的網站，選單出現「加到主畫面」，裝起來像原生 App。

什麼是 PWA？

Progressive Web App

PWA 是「可安裝的網頁」。不用上架 App Store，使用者用瀏覽器開你的網址，就能把它加到主畫面，之後點圖示全螢幕開啟。

📇

manifest

告訴手機：App 叫什麼、用什麼圖示、什麼顏色。

⚙️

Service Worker

背景腳本，負責快取，讓 App 開得快、能離線。

提示詞

加 manifest + service worker
幫我的 Flask 網站加上 PWA：
1. static/manifest.json：名稱「亮言 · 即時翻譯」、圖示、深紫主題色、standalone。
2. static/sw.js：Service Worker，網路優先、離線用快取。
3. 在 index.html 連結 manifest、註冊 sw.js。
4. app.py 用根路徑提供 manifest.json 與 sw.js。
給我完整程式碼與放置位置。

  "name": "亮言 · 即時翻譯",
  "short_name": "亮言翻譯",
  "start_url": "/",
  "display": "standalone",

  "theme_color": "#2a1a4a",
  "icons": [

    { "src": "/static/icon-512.png", "sizes": "512x512", "type": "image/png" }

}

Service Worker（網路優先）

線上拿最新、離線用快取
// static/sw.js
const CACHE = 'liang-v1';
self.addEventListener('install', e => self.skipWaiting());
self.addEventListener('activate', e => self.clients.claim());
self.addEventListener('fetch', e => {
  const url = new URL(e.request.url);
  if(url.pathname.startsWith('/api/')) return;   // API 不快取
  e.respondWith(
    fetch(e.request).then(res => {                // 網路優先
      const copy = res.clone();
      caches.open(CACHE).then(c => c.put(e.request, copy));
      return res;
    }).catch(() => caches.match(e.request))         // 沒網路用快取
  );
});

為什麼「網路優先」？

避免更新被舊快取卡住

如果「快取優先」，你改了程式重新部署，使用者卻一直看到舊版（被快取黏住）。所以我們用網路優先：有網路就拿最新、順便更新快取；沒網路才用快取。

✅ 每次改版把 sw.js 的 CACHE 版本號 +1（v1→v2），能更確實地讓使用者換到新版。

連結 + 註冊

在 index.html 與 app.py
<!-- index.html <head> -->
<link rel="manifest" href="/manifest.json">
<meta name="theme-color" content="#2a1a4a">
<!-- 頁尾 -->
<script> if('serviceWorker' in navigator) navigator.serviceWorker.register('/sw.js'); </script>
# app.py：從根路徑提供
from flask import send_from_directory
@app.route('/manifest.json')
def manifest(): return send_from_directory('static','manifest.json')
@app.route('/sw.js')
def sw():
    r = send_from_directory('static','sw.js')
    r.headers['Service-Worker-Allowed'] = '/'
    return r

實測安裝

手機加到主畫面
  1. 手機瀏覽器開你的網站（本機測需同網段；上線後用網址最準）。
  2. 瀏覽器選單 → 「加到主畫面 / Add to Home Screen」。
  3. 回主畫面，點圖示，App 全螢幕開啟。
