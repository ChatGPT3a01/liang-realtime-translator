先搞懂：Render 是什麼？

別急著操作，先懂它在做什麼

Render 是一台「幫你 24 小時開著、專門跑你程式」的雲端電腦。你把 App 交給它，它就一直幫你執行，並給你一個永久網址，全世界都連得到。

🏠 打個比方：你的 App 到目前為止都只在你自己的筆電上跑（python app.py）。但筆電一關機、或你人不在家，App 就斷線了。Render 就像幫你租一台永遠不關機的電腦放你的 App——你去睡覺、去日本，它照樣在線上。
💡 這類「幫你跑後端程式 + 給你網址」的雲端平台有很多（Render、Railway、Fly.io…），我們挑 Render，因為它免費方案夠用、用 GitHub 登入、設定最簡單。

為什麼不能像簡報一樣直接丟上去？

「靜態檔案」 vs 「會執行的伺服器」

你的 App 分成兩半，關鍵差別在後端：

🖼️

前端＝靜態檔案

畫面（HTML／CSS／JS）。像一張海報，放著就好、不會動。放哪都行。

🐍

後端＝會執行的程式

app.py 是 Python 伺服器，要一直執行：收翻譯請求、呼叫 Gemini、查天氣、保管金鑰。

因為你的 App 有一個需要「一直執行 Python」的後端，就不能丟到只會放靜態檔的地方——必須放在「能跑 Python」的主機，也就是 Render。

🧭 今天的實例：老師的「教學簡報」是純靜態檔，所以放 Netlify；但這支「翻譯 App」有 Python 後端，要放 Render。兩個平台用途不同，別搞混囉。

Render 幫你做的三件事

你只要按幾個鈕，它自動完成
📥

① 抓程式碼

從你的 GitHub 倉庫自動下載最新程式。

📦

② 裝套件

照 requirements.txt 跑 pip install，把 Flask 等裝好。

🚀

③ 跑起來給網址

執行 python app.py，給你 https://xxx.onrender.com。

📄 這三件事其實就寫在待會的 render.yaml 裡（buildCommand＝裝套件、startCommand＝跑起來）。看不懂沒關係，Render 會照著做。

① 上傳前：千萬別把 .env 傳上去

金鑰外流事小，帳單爆掉事大
🚨 網頁上傳的人特別注意：用「拖檔上傳」時，.gitignore 不會幫你擋 .env！（.gitignore 只在用 Git 指令時才生效。）所以拖檔前，手動把 .env 這個檔挑掉、不要拖進去。
🔑 為什麼這麼嚴重：.env 裡是你的 Gemini／Tavily 金鑰。傳上 GitHub 等於公開給全世界，可能被拿去亂用、把你的額度或帳單燒爆。金鑰正確的家是 Render 儀表板（下一步就教）。
✅ 上傳後自我檢查：到 repo 的檔案列表看看有沒有 .env。有的話 → 點進去 → 右上角垃圾桶 → Delete 刪掉。（.env.example 是空白範例、沒有金鑰，留著沒關係。）
⌨️ 會用 Git 指令的人：git init → git add . → git commit → git push 即可，此時 .gitignore 會自動排除 .env。不熟就用上面的網頁上傳法，或把這幾行貼給 AI 請它教你逐步操作。

② 設定環境變數（關鍵！）

金鑰放這裡，不進程式碼

往下找到 Environment Variables，把金鑰一組一組加進去：

KeyValue
GEMINI_API_KEY你的 Gemini 金鑰
TAVILY_API_KEY你的 Tavily 金鑰（選填）
OPENAI_API_KEY你的 OpenAI／Groq／DeepSeek 金鑰（選填；填了整站訪客免自填就能用 OpenAI 相容服務）
🔐 這就是為什麼程式用 os.getenv()——它會來這裡讀，金鑰不出現在程式碼或 GitHub。
💡 加了 OPENAI_API_KEY 這步驟跟 Gemini 完全一樣：New → Key/Value → Save。填了它，App 選「OpenAI 相容」時金鑰欄可留空（伺服器共用）。⚠️ OpenAI 是付費 API，記得去 OpenAI 帳戶設 Usage limit。

免費方案要知道

  • 免費服務閒置會休眠，第一次連線要等 30–60 秒喚醒，屬正常。
  • 喚醒後就正常快。想常駐可升級付費方案。

之後怎麼更新？

push 就自動部署
改程式 → 上傳/push 到 GitHub
        → Render 偵測到變更
        → 自動重新部署（約 2–5 分鐘）
        → 網址上就是新版

改了看不到？PWA 有快取——把 sw.js 的版本號 +1、手機重新整理兩次。

常見部署問題

上不了線先看這

Build 失敗？

多半是 requirements.txt 少套件或打錯字，看 log 補上。

啟動就掛？

確認 Start Command、app.py 用環境變數 PORT。

功能沒反應？

檢查環境變數金鑰有沒有填、有沒有拼錯。

第一次好慢？

免費方案喚醒中，等 30–60 秒。
